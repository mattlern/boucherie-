# Images mises à jour

Les nouvelles images utilisées par le code sont rangées dans :

`assets/images/nouvelle-selection/`

Le code HTML pointe maintenant vers des noms propres et stables, par exemple :

- `accueil-boucherie.png`
- `accueil-charcuterie.png`
- `accueil-traiteur.png`
- `boucherie-brochettes.png`
- `traiteur-plats-cuisines.png`
- `logo-officiel.png`

Pour GitHub : garde exactement ce dossier et ces noms de fichiers.  
Si tu veux remplacer une image plus tard, il suffit de remplacer le fichier PNG correspondant par une nouvelle image avec le même nom.

Le logo officiel utilisé dans le header et le footer est :

`assets/images/nouvelle-selection/logo-officiel.png`
